# Synapse — website

The marketing site + developer docs for **[Synapse](https://github.com/andrecorugda/synapse-app-builder)**, the app builder for Laravel + Filament.

- `index.html` — the landing page, **built in Synapse and exported to static** (`php artisan ai-page-builder:export`).
- `docs/` — the developer documentation.

Hosted free on GitHub Pages. Built with Synapse. 🧠
